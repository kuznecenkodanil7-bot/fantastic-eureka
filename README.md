# Chat Reminder для Minecraft 1.21.11 Fabric

Клиентский Fabric-мод: пишет настраиваемое сообщение в чат через заданный интервал.

По умолчанию:
- сообщение появляется каждые 4 минуты;
- первое сообщение появляется через 10 секунд после входа в мир/сервер;
- сообщение показывается локально у игрока в чате, без отправки команды/текста на сервер.

## Как собрать JAR

Нужно установить Java 21 и Gradle 9.x.

В папке проекта выполнить:

```bash
gradle build
```

Готовый файл будет здесь:

```text
build/libs/chat-reminder-fabric-1.0.0.jar
```

Его нужно положить в:

```text
.minecraft/mods
```

## Как изменить сообщение и время

После первого запуска появится файл:

```text
.minecraft/config/chat_reminder.json
```

Пример конфига:

```json
{
  "enabled": true,
  "message": "&a[Авто-сообщение] &fЭто сообщение отправляется каждые 4 минуты.",
  "intervalSeconds": 240,
  "firstDelaySeconds": 10
}
```

Что менять:
- `message` — текст сообщения;
- `intervalSeconds` — интервал отправки в секундах;
- `firstDelaySeconds` — задержка первого сообщения после входа;
- `enabled` — включить/выключить мод.

Цвета можно писать через `&`-коды Minecraft:

```text
&a зелёный, &b голубой, &c красный, &e жёлтый, &f белый, &r сброс
```

Файл конфига можно менять прямо во время игры. Мод проверяет его примерно раз в секунду и сам перезагружает настройки.

## Версии

Проект настроен под:

```text
Minecraft: 1.21.11
Yarn: 1.21.11+build.4
Fabric Loader: 0.18.4+
Java: 21+
```

Fabric API не нужен.
